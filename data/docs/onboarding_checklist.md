# Onboarding Checklist

## Day 1
- [ ] Receive and sign employment contract
- [ ] Complete I-9 / right to work verification
- [ ] Set up laptop and connect to Amiseq-Corp WiFi
- [ ] Activate company email
- [ ] Set up MFA on all systems
- [ ] Meet your onboarding buddy
- [ ] Attend welcome session with HR
- [ ] Join Slack and introduce yourself in #general

## Week 1
- [ ] Complete VPN setup
- [ ] Set up direct deposit via HiBob
- [ ] Review Employee Handbook
- [ ] Meet your team and manager 1:1
- [ ] Get badge access from office admin
- [ ] Complete security awareness training
- [ ] Request software access (GitHub, Jira, Figma)

## Month 1
- [ ] Complete role-specific onboarding training
- [ ] Attend department all-hands meeting
- [ ] Submit first expense report (if applicable)
- [ ] Complete 30-day check-in with HR
- [ ] Set 90-day goals with your manager
